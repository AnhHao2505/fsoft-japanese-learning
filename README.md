# FPT Software - Japanese Learning Web App

Ứng dụng web ghi lại và hiển thị quá trình học tiếng Nhật tại FPT Software.

## Tech Stack

- **Frontend:** React + Vite (Vanilla CSS)
- **Backend:** Spring Boot (Java)

## Getting Started

### Prerequisites
- Node.js >= 18
- Java 17+
- Maven

### Run Frontend
```bash
cd frontend
npm install
npm run dev
```

### Run Backend
```bash
cd backend
./mvnw spring-boot:run
```
